**APP功能**   
对各休闲 APP 用悬浮窗遮住首页的推荐内容，但顶部搜索栏、底部其他菜单不做限制。  
即保留了“搜索引擎”的功能同时，节制了漫无目的的沉迷。
想关闭悬浮窗就得做一道难度适中的算术题。  

**适用范围**  
小红书、知乎、抖音、B 站……，预置支持 6 个APP，还可手动添加其他 APP。
